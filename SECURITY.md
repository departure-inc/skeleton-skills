# Security Policy

## Supported Versions

このリポジトリは `main` ブランチの最新版のみをサポート対象とします。

## Reporting a Vulnerability

脆弱性を発見した場合は、公開 Issue ではなく GitHub の [Security Advisories](https://github.com/departure-inc/skeleton-skills/security/advisories/new) から報告してください。

報告には以下を含めてください。

- 影響を受けるスキル / ファイル
- 再現手順
- 想定される影響範囲

報告を受け取り次第、内容を確認し対応方針を折り返しでお知らせします。
